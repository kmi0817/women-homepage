# women-homepage
한국여성의집

1. Installation
```
npm install express
npm install ejs
npm install method-override
npm install --save-dev nodemon
npm install mysql
npm install http
npm install crypto --save
npm install sanitize-html // XXS 관련 보안 라이브러리
npm install xlsx
```

2. 데이터베이스 연동
- config 디렉터리의 .env 파일 생성 후 아래 내용 입력
```
DATABASE_SPRINT_PASSWORD=데이터베이스 비밀번호
```
