## Naming Rule
### 1. GET
- show[One] in \<Table\>
- get\<Table(s)\>[By...] in \<Table\>Storage

### 2. POST
- reigster in \<Table\>
- save in \<Table\>Storage

### 3. PATCH
- modify in \<Table\>
- update in \<Table\>Storage

### 4. DELETE
- remove in \<Table\>
- delete in \<Table\>Storage
